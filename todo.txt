Github gebruikt een credential manager om via tokens met github te praten.
Je moet read / write naar deze credential manager. Per gebruiker kan je dan in
een .env file de gerelateerde key:value pairs opslaan in de vorm van USER:ACCESS_TOKEN.

1. externe PS1 scripts met (PARAMETER INPUT VANAF JS FILE) en (OUTPUT IN JS FILE IN VORM VAN JSON).
2. Error handling voor deze PS1 scripts
3. Verwijder alle gevonden github credentials
4. Voeg nieuwe github credentials toe, op basis van DESIRED_USER object in app.js

--> als PS1 te ingewikkeld is, try catch toevoegen in credential_helper.js

TIPS:

package.json --> "bin" om script te specificeren en daarna globaal installeren via npm
